About This Project

This project is designed to streamline the creation of structured and SEO-optimized content across multiple platforms like blogs, social media, and email campaigns. It leverages advanced AI models and intuitive user prompts to deliver JSON-formatted drafts tailored for professional use.

Key Features:

  • Support for multiple content formats: Blog posts, Instagram posts, LinkedIn updates, and emails.
  • Customizable prompts for tailored outputs aligned with specific needs.
  • Automatic error correction and JSON validation for reliable results.

Technology Stack:

  • Backend: Flask framework for handling web requests and user inputs.
  • AI Integration: Groq API for generating AI-driven content.
  • Frontend: HTML, CSS, and JavaScript for a responsive user interface.
  • Environment Management: dotenv for secure API key storage.

This project demonstrates the potential of AI in content creation, offering a user-friendly interface for generating high-quality, platform-specific drafts.
